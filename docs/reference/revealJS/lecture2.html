Tip1: Name of Variables

  • A variable name should start with an alphabetic character (like a, b, c, etc.) and can include letters, numbers, and underscores _. It must be all one word with no spaces.
  • You can’t use any of the keywords or reserved words as variable names in Java (for, if, class, static, int, double, etc).
  • Use meaningful variable names: Start variable names with a lower case letter and use camelCase.
  • Variable names are case-sensitive and spelling sensitive.
  • Never put variables inside quotes ("").
